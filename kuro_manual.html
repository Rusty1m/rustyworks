Kuro 取説 / Manual

Kuro は、正確な色で観るための macOS 用ネイティブ動画プレーヤーです。libmpv による幅広い形式の再生に、Metal によるリアルタイムのカラーグレーディングとモニタリングを組み合わせています。
Kuro is a native macOS video player built for accurate color, combining wide-format playback via libmpv with real-time color grading and monitoring powered by Metal.

概要 / Overview

Japanese

Kuro は、1つのウインドウで「再生・色補正・解析」を行う動画プレーヤーです。リファレンス（Rec.709 / Gamma 2.4）に合わせ込んだ一次補正を、Metal でリアルタイムに適用します。

正確に見て、思いどおりに整える。軽快な再生と、プロ志向のカラーツールを両立しています。

English

Kuro is a video player that handles playback, color correction, and analysis in one window. It applies a reference-matched (Rec.709 / Gamma 2.4) primary grade in real time with Metal.

View accurately, shape it your way — fast playback paired with pro-grade color tools.

動作環境 / Environment

日本語

  • Apple Silicon（M1 以降）専用
  • Mac App Store で配布
  • 入力：ローカルファイル＋直接URL（http(s) / HLS）

English

  • Apple Silicon only (M1+)
  • Distributed on the Mac App Store
  • Input: local files and direct URLs (http(s) / HLS)

クイックスタート / Quick Start

Japanese

  1. 動画ファイルをウインドウにドロップ、または ⌘O で開きます。
  2. Space で再生 / 一時停止。
  3. 補正パネルを開きます（表示メニュー → 補正パネル、⇧⌘C）。
  4. 自動調整 を押すと、ホワイトバランスとレベルを自動で整えます。
  5. さらに整えたいときは 基本 / カラー / 画質 タブで手動調整します。

English

  1. Drop a video onto the window, or open it with ⌘O.
  2. Press Space to play / pause.
  3. Open the Correction panel (View → Correction Panel, ⇧⌘C).
  4. Press Auto Adjust to set white balance and levels automatically.
  5. Fine-tune manually in the Basic / Color / Quality tabs.
まずは 自動調整 / Auto Adjust から。結果はそのまま手動で微調整できます。
Start with Auto Adjust — the result stays fully editable.

画面概要 / Window Overview

  • 映像表示・ドラッグ&ドロップ / Video view, drag & drop
  • 表示モード切替 ⌘G（フィット / フィル / 実寸 / 画面サイズ）

  • 再生・シーク・コマ送り/戻し・音量 / Play, seek, frame step, volume
  • 字幕の表示切替 S / Toggle subtitles

補正パネル / Correction Panel

  • 4タブ：基本 / カラー / 画質 / 音声
  • カーブ・LUT・ヒストグラム / Curves, LUT, histogram

スコープ・解析 / Scopes

  • 波形 / ベクトル / パレード / ヒストグラム
  • スポットプローブ（画素値の正確な測定）/ Spot probe

インスペクタ / Inspector

  • 映像 / 音声 / 性能 の3タブ / Video, Audio, Performance
  • 解像度・FPS・コーデック・HDR状態 など

設定 / Settings

  • 言語（日/英）・表示色域・表示ガンマ
  • アップスケール・Kuro PRO

主な機能 / Main Features

カラーグレーディング / Color Grading

  • リフト / ガンマ / ゲイン、オフセット、コントラスト、3-Way
  • 色温度 / ティント、色相 / 彩度、トーンカーブ
  • Lift/Gamma/Gain, Offset, Contrast, 3-Way, WB, Hue/Sat, Curves

3D LUT

  • .cube ディスプレイ LUT の読み込み
  • 現在のグレードを .cube として書き出し
  • Load / export .cube LUTs

アップスケーリング / Upscaling

  • MetalFX / Anime4K / Apple AI 超解像
  • 低解像度の映像をリアルタイムで高精細に

HDR 再生 / HDR Playback

  • ディスプレイの実輝度に合わせてトーンマップ
  • Tone-mapped to your display's real brightness

自動調整 / Auto Adjust

  • ワンクリックで WB＋レベルを自動補正（無料）
  • One-click white balance + levels (free)

オーディオ / Audio

  • 音声トラック選択・A-V 同期調整・ナイトモード
  • Track select, A/V sync, night mode

基本的な使い方 / Workflow

Japanese

  1. 再生して全体を確認します（終端で再生を押すと頭から再生し直します）。
  2. 自動調整 で土台を作ります。
  3. カラー タブで色温度 / ティント、3-Way、彩度を微調整。
  4. AWB / AE のスポイトで、画面の一点を基準に WB / 露出を合わせます（PRO）。
  5. 仕上がった補正を .cube として書き出し、他ツールでも使えます（PRO）。

English

  1. Play through to review (pressing play at the end restarts from the start).
  2. Build a base with Auto Adjust.
  3. Fine-tune temperature / tint, 3-Way, and saturation in the Color tab.
  4. Use the AWB / AE eyedroppers to set WB / exposure from a point on screen (PRO).
  5. Export your grade as a .cube to reuse in other tools (PRO).

Kuro PRO

Kuro は無料で使えます。Kuro PRO（買い切りのアプリ内課金）で、プロ向けのカラーツールを解放します。 / Kuro is free to use. Kuro PRO (a one-time in-app purchase) unlocks the pro color tools.

無料 / Free

  • 再生全般、自動調整、シーン毎の自動調整（バランス）
  • スコープ、スポットプローブ、インスペクタ
  • 字幕、プレイリスト、音声タブ、各種フィット・再生速度

Kuro PRO

  • 手動カラー補正（基本 / カラー）、画質（NR / シャープ）
  • AWB / AE スポイト、A/B 比較、シーン自動の暗部強調
  • .cube LUT 書き出し

注意事項 / Notes

Japanese

  • プライバシー：データを一切収集しません。接続するのは、あなたが入力した URL のときだけです。
  • 直接の動画 URL（http(s) / HLS）に対応。YouTube などのサイト URL には対応していません。
  • Kuro はプレーヤーです。映像そのものの書き出しは行いません（グレードは .cube として書き出せます）。

English

  • Privacy: Kuro collects no data. It uses the network only when you open a video URL yourself.
  • Supports direct video URLs (http(s) / HLS). Website URLs such as YouTube are not supported.
  • Kuro is a player; it does not export video itself (you can export the grade as a .cube).

FAQ

対応形式は？ / Supported formats?

MP4 / MOV / MKV / WebM / AVI / FLV / WMV / TS / M2TS / HEVC / AV1 など、主要な動画形式に幅広く対応します。 / Broad support: MP4, MOV, MKV, WebM, AVI, HEVC, AV1 and more.

Intel Mac で動く？ / Runs on Intel?

いいえ。Apple Silicon（M1 以降）専用です。 / No — Apple Silicon (M1+) only.

字幕は使える？ / Subtitles?

はい。トラック選択と表示切替（S）に対応します。 / Yes — track select and toggle with S.

オフラインで使える？ / Works offline?

はい。ローカルファイルの再生・補正は完全にオフラインで動作します。 / Yes — local playback and grading work fully offline.
